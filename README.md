# STUDENT_PERFORMANCE_EDA
Student Performance EDA Project

This project analyzes student performance data using Python and Exploratory Data Analysis (EDA).

Tools Used:
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

Analysis Performed:
- Univariate Analysis
- Bivariate Analysis
- Correlation Heatmap
- Pairplot Visualization

Insights:
- Higher study hours improve grades.
- Attendance affects performance.
- Subject scores are positively correlated.
